# Lab 1 — Kotlin Basics

## Description
Ce TP a pour objectif de pratiquer les opérateurs mathématiques et l’utilisation des variables en Kotlin.

## Contenu
- Opérations arithmétiques (`+ - * /`)
- Priorité des opérations et parenthèses
- Calcul d’un total et d’une moyenne
- Moyenne en `Double`
- Bonus sur le score final

## Exécution
Le programme s’exécute dans `fun main()` et affiche les résultats dans la console.
